A full-stack logic implementation for real-time taxi booking. This system calculates the most 
efficient route between two coordinates and generates a dynamic price estimate based on distance, 
vehicle type.

Dynamic Pricing Engine: Calculates cost using a multi-variable formula:Cost = BaseFare 
+ (Distance x Rate)+ (TrafficDelay x Surge)
